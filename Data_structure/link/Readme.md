## Single linkedlist
링크드 리스트는 head와 tail의 정보가 필요하다.
* head : 리스트의 시작 주소를 담고 있어야 한다.
    - head가 리스트를 제대로 가리키지 못하면 리스트를 찾을수 없다.
* tail : 리스트의 끝 을 알려야 한다.
* traversing : 리스트의 node들을 움직이기 위해 순회 알고리즘 필요하다
    - link hopping or pointer hopping이라고 불린다.
